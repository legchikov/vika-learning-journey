# Prepare Conda env (single run)
installing link
 https://docs.conda.io/projects/conda/en/latest/user-guide/install/windows.html

Create env
```
conda create -n tinyproj python=3.9
```

Activate env
``` 
conda activate tinyproj
```

Install pytest
```
pip install -U pytest
```

Check pytest
```
pytest --version
```
