# Проект №0 "Hello, World"

Задание
```
- Создать ветку в репозитории
- Написать программу hello.py которая выводит hello world
- Оформить hello world как функцию (передается аргумет name, функция выводит "Hello, <name!>")
- Сделать чтобы <name> всегда выводился с большой буквы. Если в <name> только цифры или спецсимволы, вывести ошибку, что такое имя не подходит.
- Сделать тест кейсы для функции
- Написать тесты с помощью pytest
- Сделать чтобы имя передавалось как параметр при запуске - python hello.py --name Vika
- Сделать PR в репозиторий
- Сделать документацию по скрипту, как запускать, какие параметры есть, пример запуска
- Сделать PR в репозиторий
- После принятия PR удалить ветку

```

# Алгоритм работы

```
1. Создать простую функцию для вывода Hello, World!
2. Добавить параметр к функции с помощью  get_args()
3. Написать правила для валидации параметра и вывода
4. Добавить функцию main()
5. Установить pytest
6. Написать тесты для функции
7. Провести тестирование

```

# Using hello.py
